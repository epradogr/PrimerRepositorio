De lo virtual a lo físico

Para imprimir un objeto con ayuda de una impresora 3D la realidad aumentada tiene mucha utilidad, en clase revisamos el proceso de usar polycam para tener el objeto de forma virtual, de ahí se convierte a un tipo de archivo que lo pueda leer la computadora para que posteriormente se pueda procesar esta informacion con un programa e imprimirlo en 3D.
