Pasos para desarrollar mi programa en C++

  1. Declarar las librerías string y iostream.
  2. Declarar las variables de intentos maximos, la palabra a encontrar, numero de letras, letras encontradas, banderas e intentos.
  3. Se cuentan las letras de la palabra clave que es "sistemas" y se imprime esa cantidad pero en forma de guiones, es decir, se imprimen ocho guiones.
  4. Se pide ingresar una letra, si la letra forma parte de la palabra, se sustituye en el espacio del guión, si la letra no se encuentra se resta un intento.
  5. El programa se itera hasta que el numero de intentos llegue a cero o hasta que el jugador encuentre la palabra, se imprimirá el mensaje respectivo de cada resultado.
