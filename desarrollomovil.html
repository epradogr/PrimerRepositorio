Desarrollo móvil.

FlutterFlow

Este fue el lenguaje de programación que se usó para desarrollar la interfaz de mi aplición móvil. Combina utilidades fáciles con programación más avanzada. Click aquí para crear tu cuenta.

Firebase

Firebase es una herramienta de backend de Google para el desarrollo de tu app móvil, desde FlutterFlow se puede enlazar fácilmente a tu proyecto. No omitas este paso, ya que sin backend, tu app no tendrá respaldo.

Enlazando Firebase

Para enlazar Firebase con tu app móvil de Flutterflow, tras crear un nuevo archivo le vas a dar click al botón de Create project. Lo único que tendrás que hacer es escoger la región más cercana a ti y una cuenta de Google.

Ya tienes Firebase

Tras esperar de 3 a 5 minutos, se generará el enlace, solo tendrás que activar la opción de Enable Authentication y ya podrás darle click a Start Building para crear tu app móvil. Haz click en cada imagen de abajo para ver mi app.
