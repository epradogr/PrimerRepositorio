Render.

Crea tu cuenta

Para empezar a usar render, tienes que crearte una cuenta con tu correo electrónico. A través de este link puedes hacerlo.

Instala Gunicorn

Dentro de tu entorno virtual, va a ser necesario que instales la biblioteca de gunicorn para poder subir tu servidor a render. Lo podrás hacer siguiendo los pasos de la imágen, iniciando por el comando pip install gunicorn.

Primer Proyecto

Da click en crear nuevo proyecto y ponle un nombre. Después seleccionaras la opción de web services para empezar.

Selecciona un repo

Render te dara varias opciones para hacer un deploy, usa la opción que prefieras, yo te recomiendo usar un repo desde GitHub.

Sube tu repo

Después de haberlo seleccionado, elige la opción que decidas ya sea de paga o gratis, después dale click al botón de iniciar Deploy y comenzará a subirse tu server.

¡Listo!

De haber quedado todo bien, te saldrá una palomita verde con el mensaje de Live y te dará un link. Puedes acceder a mi link aquí. Ten en cuenta que al ser un server gratis, tomará unos segundos en cargar por completo.
