Backend.

Crea tu entorno virtual

Si usas windows, abre el cmd y dentro de una carpeta usa el comando "python -m venv venv". Si lo hiciste correctamente se abrirá una subcarpeta llamada venv, ese es tu entorno virtual.

Actívalo

Para activarlo escribe después de lo anterior "venv\Scripts\activate" si lo hiciste correctamente abajo se pondrá una burbuja de (venv), ya está activado. Esto repítelo cada vez que lo quieras abrir.

¿Para que sirve un entorno virtual?

Servirá como un espacio aislado en tu computadora en el cuál podras instalar bibliotecas, dependencias y lo que necesites para tu proyecto, pero solo dentro de esa burbuja, no afectará tu entorno exterior de trabajo,. En caso de que instales la versión más reciente de python pero tu proyecto está en una versión anterior, no tendrás ningún problema si tu proyecto está dentro de un entorno virtual.

Flask

Flask es una biblioteca que utilizarás para crear tu backend, para instalarla escribe "pip install Flask".

Checa las bibliotecas

Para ver las bibliotecas que instalaste, escribe "pip freeze". Esto te dará una vista de lo que tienes instalado.

Función principal

Abre python desde tu carpeta y escribe el siguiente código:

  • from flask import Flask: esto permitirá utilizar recursos de la biblioteca Flask
  • app = Flask(__name__): será útil para localizar recursos.
  • @app.route('/'): asocia un URL con la función home.
  • def home(): se define una función llamada home, se podrá acceder con el URL raíz de la aplicación.
  • return "Mi primer backend": La función home abrirá el navegador y se expondrá el texto escrito.
  • if __name__=='__main__': el servidor web se ejecutará al ejecutar el archivo python.
  • app.run(debug=True): hará los cambios automáticos en el navegador.

Ejecuta el programa

Dentro de la consola excribe "python app.py". Esto ejecutará el programa y te dará el link http://127.0.0.1:5000/ Ábrelo en tu navegador y encontrarás lo que escribiste en return. Esto activa tu servidor. para cerrarlo presiona CTRL+C.

Así se ve en el navegador, recuerda que cualquier cambio que hagas en tu proyecto y guardes, se verá reflejado automáticamente en tu servidor.

Coloca archivos html

Para poder colocar archivos html haz lo siguiente:

  1. Crea una carpeta templates donde pongas todos tus archivos html.
  2. Después de la linea import Flask pon: ", return_template".
  3. En el return de la función home pondrás render_template('') Dentro de las comillas del paréntesis pon el nombre de tu archivo.
  4. Ya con esto estas llamando la ruta de tu archivo, para agregar más links a tu server, copia la función app route, y solo cambia la ruta.

Agregar imagenes

Para poder colocar imagenes haz lo siguiente:

  1. Crea una carpeta static, adentro una carpeta images. Aquí irán todas tus imgaenes.
  2. Dentro del archivo html pondrás como ruta lo que se muestra en la captura de pantalla. Solo cambiarás el nombre de la imagen.
  3. Listo, haz esto con todas las imágenes que necesites.

Agrega un link a otra página

Siempre que quieras agregar un link de una página a otra haz lo siguiente:

  1. Asegúrate de que el archivo html se encuentre dentro de la carpeta templates.
  2. Asegúrate de que también le hayas hecho una ruta desde python.
  3. Para que funcione, dentro de tu archivo html pondrás la ruta que se muestra en la captura de pantalla, solo cambiarás el nombre de tu archivo.

Estas herramientas son más que suficientes para que puedes tener una página web con un servidor virtual, a partir de aquí puedes crear lo que quieras, no olvides hacer git push.
