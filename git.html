Cómo hacer tu repositorio en git

Paso 1

Descargar git desde Git descargar.

Paso 2

Crear tu repositorio en GitHub.

Paso 3

Clonar tu repositorio en git bash usando "git clone".

Paso 4

Una vez clonado, puedes trabajar en tu repositorio para almacenar una versión de este cuando quieras.

Paso 5

Para almacenar cada versión, hay tres sencillos comandos que tienes que poner en Git Bash:

  1. git add .
  2. git commit -m "mensaje"
  3. git push

Paso 6

¡Listo! ya puedes revisar todas las versiones que guardes a través de git en GitHub.
